# Computer-Aided Digital Systems Projects

This repository contains five academic projects completed for the Computer-Aided Design of Digital Systems course.

## Projects

| Project | Title | Description |
|---|---|---|
| CA1 | RTL Hash Generator | RTL design of a hash generator using datapath and controller |
| CA2 | Matrix-Vector Multiplication | Hardware design for matrix-vector multiplication |
| CA3 | Structural Hash Generator | Structural implementation using basic FPGA cells |
| CA4 | Yosys Actel Mapping | Synthesis and mapping to Actel-style cells |
| CA5 | Scheduling and Verilog Generator | Scheduling algorithms and automatic Verilog generation |

## Technologies

- Verilog HDL
- RTL Design
- FSM Design
- Datapath and Controller Design
- Python
- Yosys
- ModelSim
